# Linkurate-
Linkurate adalah aplikasi website berbasis Single Page Application (SPA) untuk memeriksa keamanan tautan secara cepat dan sederhana guna mencegah korban phishing dan scam. Aplikasi ini menganalisis pola tautan mencurigakan bagi pengguna umum, pelajar, dan pelaku UMKM yang membutuhkan solusi praktis tanpa proses rumit.
